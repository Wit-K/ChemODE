# ChemODE
Official implementation for the paper "Robust Estimation of Multi-Scale and Non-Linear Reaction Networks via Feature-Normalized Sparse Neural ODEs".

## Usage
1. Open the notebook `ChemODE_Final.ipynb` in Google Colab.
2. Set Runtime to **T4 GPU**.
3. Run all cells to reproduce Figures 1-3 from the manuscript.

## Dependencies
- torchdiffeq
- pysindy
- torch
- numpy
